# Emnm

Convert numbers to GitHub emoji numbers.

## Installation

Add this line to your application's Gemfile:

    gem 'emnm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install emnm

## Usage

```
$ emnm ruby 2.1.1
ruby :two:.:one:.:one:
```
